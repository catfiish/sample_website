Important HTML tags

This page is designed as a quick reference for some common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make html tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the important ones.

Block Elements

Tag Name Purpose
ol ordered list make an ordered list
ul unordered list make an unordered list
header header include a container for introductory content
div division group elements to format them with CSS
h1‐h6 headings include a heading (levels 1‐6)
p paragraph include a paragrah of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header section define a section or page header like a div
blockquote block quote represent a long quotation
